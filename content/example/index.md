---
title: "A page with an image"
description: "Example showing how images work in Threshold."
---

Drop a photo into your page's `images/` folder and reference it with a normal
Markdown image — Threshold automatically makes it responsive, fast, and
privacy-safe (it even strips the GPS location that phones embed in photos):

![A mirror-polished sculpture reflecting a blue sky](images/sculpture.jpg)

When you want a caption, use the `image` shortcode instead:

{{< image src="images/sculpture.jpg" alt="A mirror-polished sculpture reflecting a blue sky" caption="A chrome sculpture catching the afternoon sky." >}}

You can delete this example page whenever you like.
