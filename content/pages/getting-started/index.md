You don't need to know how to code to run this site. You change it by **talking to Claude Code** — describe what you want in plain language, and it edits the files and shows you the result.

## Make a change

Just ask:

- *"Change the site name to Maria's Kitchen."*
- *"Add a page called Projects and put it in the menu."*
- *"Use a warmer color for the links."*
- *"Add this photo to my About page."*

Claude makes the edit, previews it, and tells you what changed. Don't like it? Ask it to undo.

## See your changes

Claude keeps a live preview running while you work, so the site updates as you go. Ask it to *"show me the site"* any time.

## What's in this guide

- **[Writing](/pages/writing/)** — formatting text, adding images, and emoji.
- **[Personalizing](/pages/personalizing/)** — colors, fonts, your logo, and site identity.
- **[Blog](/blog/)** — how the example blog works.

## When it feels like yours

Ask Claude to **publish** — that builds the site and pushes your changes live. And when you're done with the examples, ask it to **clear the demo content** for a clean slate.
