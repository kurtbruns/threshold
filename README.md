---

Threshold is a starter kit for building a website with [Claude Code](https://claude.com/claude-code). It contains advice, design choices, and recommendations for building a static website with Hugo baked into the context provided to Claude Code.

- Download the project folder to your computer and open it with Claude Code. Run the `/setup` skill or ask Claude what's involved with setting up the site *locally* on your computer.
